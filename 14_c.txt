Threshold Values Chosen
cv2.Canny(gray, 50, 150)
Lower Threshold = 50
Upper Threshold = 150
Justification-
The lower threshold (50) helps detect weak but meaningful edges.
The upper threshold (150) ensures that only strong edges are definitely preserved.
A 1:3 ratio between thresholds is commonly used because it balances:
noise suppression,edge continuity,and detection accuracy in real-time video streams.This combination provides stable edge detection without excessive noise.
